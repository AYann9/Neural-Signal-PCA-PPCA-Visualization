# Neural-Signal-PCA-PPCA-Visualization
## Neural Signal Processing and Machine Learning
### Dimensionality Reduction & Visualization with PCA / PPCA

This project implements **Principal Component Analysis (PCA)** and **Probabilistic PCA (PPCA)** from scratch for neural data dimensionality reduction and visualization.

## Tasks
1. Implement PCA using NumPy
2. Select optimal components with the elbow method
3. Derive & implement PPCA with EM algorithm
4. Visualize 2D / 3D projections of neural data
5. Compare PCA vs PPCA performance

## Data
Neural data provided by Prof. Krishna Shenoy’s lab at Stanford University (for educational use only).

## Files
- `solutions.ipynb`: Full code & solutions
- `realdata.mat`: Neural dataset
- `README.md`: Project description
